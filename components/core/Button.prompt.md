Neon sign button — uppercase Oswald, sharp corners, glow instead of shadow; hover brightens the glow (never scales).

```jsx
<Button variant="primary">Listen now</Button>
<Button variant="secondary" size="sm">Subscribe</Button>
<Button variant="ghost">All episodes</Button>
```

Variants: primary = pink fill (one per view, the brand moment), secondary = teal fill, ghost = hairline border. Disabled drops the glow.
